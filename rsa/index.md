### 2：计算 n 的欧拉函数 m = φ(n)。
φ(n) 表示在小于等于 n 的正整数之中，与 n 构成互质关系的数的个数。

n = P * Q，P 与 Q 均为质数，则 φ(n) = φ(P * Q)= φ(P - 1)φ(Q - 1) = (P - 1)(Q - 1) 。

m = φ(n) = 66 * 70 = 4620
